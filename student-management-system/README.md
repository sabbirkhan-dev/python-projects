# Student Management System

This is a simple **Student Management System** built using Python.
The program allows users to add, view, search, and delete student records using a command-line interface.

## Features

* Add new student
* Show all students
* Search student by ID
* Delete student by ID
* Input validation using try/except

## Technologies Used

* Python
* Lists
* Dictionaries
* Loops
* Exception Handling

## How It Works

The program runs in a loop and asks the user to choose an option.

Available options:

* **add** → Add a new student
* **show** → Display all students
* **search** → Search student by ID
* **delete** → Delete student by ID
* **break** → Exit the program

## Example

```
Choice: add, show, search, delete, break:
```

Example student data structure:

```
{
  "name": "Sabbir",
  "id": 101,
  "class1": 10,
  "cgpa": 4.50
}
```
